# CarPurchaseXGBoost

Projeto de Machine Learning para prever propensão à compra de carros utilizando XGBoost.

## Etapas do projeto
- Análise e tratamento de dados
- Encoding de variáveis categóricas
- Análise de correlação
- Modelagem com XGBoost
- Avaliação com ROC-AUC e matriz de confusão
- Feature importance

## Tecnologias
- Python
- Pandas
- Scikit-learn
- XGBoost
- Matplotlib / Seaborn
